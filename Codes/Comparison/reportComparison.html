  4. Sex: 75% women in netquest, 50% in prolific
  5. Financial literacy: lower in netquets
  6. Education: lower in netquets

Synthetic Netquest

A subsample from the Prolific dataset where participants match the same demographic distributions as those in the Netquest experiment.

Would Synthetic Netquest data produce the same results as Netquest?

The synthetic dataframe is created by sampling observations from the Prolific experiment without replacement. The process is as follows:

  1. Select a row from the Netquest dataframe.
  2. Find a matching row in the Prolific dataframe that has identical values in the specified columns (*selected_columns*) to the selected Netquest row.
  3. Add the matching Prolific row to the synthetic dataframe. The selected Prolific row cannot be used again in the sampling process.
